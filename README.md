# 📱 Pokédex App (React Native)

A modern Pokédex mobile application built using **React Native + Expo Router**, powered by the **PokéAPI**.

---

## 🚀 Features

- 🔍 Real-time Pokémon search (instant filtering)
- 📱 Clean card-based UI (dark mode with pink accent)
- 🧠 Optimized API usage (fast loading)
- 📊 Pokémon detail page with:
  - Image
  - Type
  - Description
  - Stats (HP, Attack, Defense)
- 🔄 Navigation using Expo Router

---

## 🛠️ Tech Stack

- React Native
- Expo
- Expo Router
- TypeScript
- PokéAPI (https://pokeapi.co/)
